## 📌 Project Overview

The **Support Ticket Management System** is a **pure Java console application** created to demonstrate core Java skills and understanding of programming concepts.  
This project is **not connected to any database or frontend**; it is designed to showcase knowledge in:

- Java Basics & Syntax  
- Object-Oriented Programming (OOP)  
- Collections Framework  

The system simulates a support ticket workflow entirely in memory using Java classes and collections.

---

## 🧰 Skills Demonstrated

### 1. Java Basics & Syntax
- Variables, data types, and operators  
- Conditional statements (`if`, `switch`) and loops (`for`, `while`)  
- Methods and parameter passing  
- Exception handling  

### 2. Object-Oriented Programming
- **Classes and Objects**: Modeling tickets and agents  
- **Encapsulation**: Private fields with getters/setters  
- **Inheritance**: Extending classes where appropriate  
- **Polymorphism**: Method overloading and overriding  
- **Abstraction**: Abstract classes and interfaces  
- Example: `Ticket`, `SupportAgent`, `TicketManager`  

### 3. Collections Framework
- `ArrayList`, `HashMap`, and `HashSet` usage for ticket management  
- Iteration with loops and iterators  
- Sorting and filtering in collections
